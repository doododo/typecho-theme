# Single
一个简洁大气，含夜间模式的 Typecho 博客模板。

## 使用方法
1. Star 本项目
2. Clone 本项目
3. 上传本主题，并放置在 `usr/themes/` 目录下
4. 登录你的 Typecho 后台，并进行安装即可食用~
